# Express TypeScript API

Simple backend API using Express.js, TypeScript, dotenv, and Router.

---

## Tech Stack

- Node.js
- Express.js
- TypeScript
- dotenv

---

## Installation

Clone the project:

```bash
git clone <your-repository-url>
cd <project-folder>
## Installation 
```bash
npm init -y
npm install express @types/express dotenv router
## Run Typescript 
```bash
npm install tsx
npx tsx src/server.ts
